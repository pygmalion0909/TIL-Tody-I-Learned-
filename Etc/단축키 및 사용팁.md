# 단축키 및 사용팁
## 1. 이클립스
### (1) 단축키
* ctrl + shift + r : 파일찾기
* ctrl + l : 원하는 라인으로 이동
* alt + shift + s : getter,setter, 오버로딩toString() 만들기
* ctrl + shift + o : import 자동생성

### (2) 라이브러리 추가
* 프로젝트에서 오른쪽 마우스 -> java Build Path클릭 -> Libraries탭 클릭 -> Add External JARs...클릭 -> 라이브러리 파일 널기 -> Apply and close
