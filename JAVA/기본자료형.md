# 기본자료형
## 1. 문자형
### (1) char
* 문자형은 char 한개 뿐
```java
char a = 'A'; // A이지만 컴퓨터는 숫자로 저장함
char a = 65; // 'A'의 10진수로 65라서 char a 에 할당 가능
```
