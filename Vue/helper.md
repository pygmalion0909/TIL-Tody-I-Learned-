# helper
## 1. 개념
* store에 있는 4가지 속성을 간편하게 코딩하게 도와주는 역할
* state는 mapState
* getters는 mapGetters
* mutations는 mapMutations
* actions는 mapActions

## 2. helper설정
```js
// App.vue
import { mapState } from 'vuex'
import { mapGetters } from 'vuex'
import { mapMutations } from 'vuex'
import { mapActions } from 'vuex'
```
* 위와 같이 불러오기
