# http 통신
## 1. 200 종류
### (1) 200번
* 요청에 대한 성공

### (2) 201번
* 요청이 성공적으로 처리되었으며 정상적으로 생성 됨
* 보통 post 요청에 대응하는 응답 상태 코드로 사용
* 201 Created로 반환
* 개발자 도구에 있는 네트워크에 headers에 Location에 보면 정확한 값을 확인 가능

## 2. 오류 종류
### (1) 400번
* 파리미터 변수값이 맞지 않는 경우
* 문법상 오류가 있어서 서버가 요청 사항을 이해하지 못하는 경우
* 파라미터를 정확하게 오타없이 보냈는지 체크 요망

### (2) 401번
* 권한이 없어서 발생함
* 인증 실패
